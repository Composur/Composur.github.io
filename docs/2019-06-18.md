## class

+ 是一种es5的语法糖
    + 类的所有方法都定义在类的prototype属性上面。
+ 静态属性
    + 指class本身的属性，class只有静态方法，没有静态属性
```
class Foo{

}
Foo.name='haha'
```
+ 私有方法和私有属性
    + 只能在类的内部访问的方法和属性，外部不能访问
+ super
    + 可以当函数使用，也可以当对象使用
        + 作为回调函数，super代表父类的构造函数，子类继承的时候必须执行一次super()
        + 作为对象的时候，子类通过super().xxx可以访问父类(除了构造函数中的)方法（但当前的this指向是子类）
